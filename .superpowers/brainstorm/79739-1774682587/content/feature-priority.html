Which features should be in v1?

Select all that belong in the first release — we'll defer the rest to v2

💬

Chat with document

Select a paper → ask questions → answers grounded in the PDF text via Qdrant chunks. Like ResearchRabbit's "Ask AI" but using your local MCP backend.

🕸️

Similarity graph

Interactive network: nodes = papers/authors, edges = semantic similarity or citation links. Filter by collection, date, tag. ResearchRabbit-style visual map.

🔍

External discovery

Search PubMed / Semantic Scholar / OpenAlex for similar papers or co-authors. Preview results, select → add directly to Zotero (syncs automatically).

🗑️

Cascade delete

Delete an item from Zotero → automatically removes Qdrant chunks, synctracker record, and Neo4j nodes. No orphan data left behind.

🩺

Library health dashboard

See which items are unindexed, have failed sync, are missing PDFs, or have duplicate metadata. Bulk-fix from one screen.

🔎

Semantic search panel

Search your library by meaning, not just keywords. Results ranked by hybrid score (dense + BM25). Click result → open in Zotero.

👤

Author network & profile

Click an author → see all their papers in your library, their co-authors, and a button to search web for more of their work to import.

⚙️

Indexing queue / status

See pending, processing, and failed indexing jobs. Retry failed items. Know what's been vectorized vs. not.
